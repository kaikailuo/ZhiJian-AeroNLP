# aeronlp-frontend

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run 
```

Notam dev
```bash
# watch and generate src/styled-system/styles.css
bun panda:css

# watch and generate src/styled-system/**.ts 
bun panda:ui

# run vite
bun dev
```

This project was created using `bun init` in bun v1.3.6. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
